Learning Attributes in HTML.

Attributes play a vital role in customising your webpage according to you
Inspect this webpage to understand the use of attributes.

1.'href' attribute

"The tag defines a hyperlink. The href attribute specifies the URL of the page the link goes to: "

2.'img', 'src', 'alt', 'width', 'height' attributes

"The tag is used to embed an image in an HTML page.
The src attribute specifies the path to the image to be displayed.
The required alt attribute for the tag specifies an alternate text for an image, if the image for some reason cannot be displayed.
The tag should also contain the width and height attributes, which specifies the width and height of the image (in pixels)"

1.'style' attribute

"The style attribute is used to add styles to an element, such as color, font, size, and more. "

1.'lang' attribute

"You should always include the lang attribute inside the tag, to declare the language of the Web page.
This is meant to assist search engines and browsers."

1.'title' attribute

"The title attribute defines some extra information about an element.
The value of the title attribute will be displayed as a tooltip when you mouse over the element"
